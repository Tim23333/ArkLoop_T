# Optional GPU acceleration for avatar template matching.
#
# Installs PyTorch with CUDA — the avatar matcher then runs as a single
# batched F.conv2d over all templates on the GPU (~8-10x faster per deploy:
# ~270ms -> ~30ms on an RTX 2070).  Without this file, the matcher uses the
# per-template cv2.matchTemplate CPU loop with identical results.
#
# Install on top of requirements.txt (or requirements_no_tesserocr.txt):
#   .venv\Scripts\python -m pip install -r requirements-gpu.txt
#
# Requires an NVIDIA GPU with a driver that supports CUDA 12.1. Packaged users
# should run ArkLoopDependencyInstaller.exe instead. The main ArkLoop bundle
# always excludes Torch, even when this requirement is installed in the build
# virtual environment.
--index-url https://download.pytorch.org/whl/cu121
--extra-index-url https://pypi.org/simple
torch==2.5.1+cu121
